
# Story-Driven Study App UI (Copy)

This is a code bundle for Story-Driven Study App UI (Copy). The original project is available at https://www.figma.com/design/3lbPuGqtCW1d6PlmvpszV3/Story-Driven-Study-App-UI--Copy-.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

## Deploy to Vercel

1. Install dependencies: `npm i`
2. Push to GitHub (see Git setup below)
3. Import project in Vercel dashboard
4. Deploy automatically

## Git Setup

If you haven't initialized Git yet:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin <your-repo-url>
git push -u origin main
```
