# 🎯 StoryStudy - Complete Feature Access Guide

## How to Access ALL Implemented Features

### Main Navigation (Always Visible in Header)

1. **📚 Flashcards** - Click the "Flashcards" button in the top-right
   - Create flashcards with front/back content
   - Study with spaced repetition (SM-2 algorithm)
   - Track retention rates and review schedules
   - Import/export flashcard decks

2. **⚙️ Settings Menu** - Click the "Settings" button in the top-right
   - Opens a comprehensive modal with 7 tabs (see below)

3. **📈 Progress** - Click the "Progress" button (appears after completing sessions)
   - View analytics dashboard
   - See focus trends over time
   - Track achievements

4. **📖 Journal** - Click the floating purple button in bottom-right corner
   - Review all session history
   - Read reflections and story outcomes
   - Export journal data

---

## 🔧 Settings Menu (Click "Settings" Button)

### Tab 1: 📚 Goals
- **Goal Setup & Management**
- Create study goals with deadlines
- Break down goals into sub-tasks
- Track progress with visual indicators
- AI-powered goal description suggestions
- Link sessions to specific goals

### Tab 2: 📅 Scheduler
- **Study Session Planning**
- Calendar view with date picker
- Schedule sessions with custom durations
- Set priorities (Low, Medium, High)
- Recurring session support
- Export to .ics (works with Google Calendar, Outlook)
- Browser notifications for reminders

### Tab 3: ⚡ Presets
- **Session Presets Manager**
- Pre-built presets:
  - Classic Pomodoro (25/5/15)
  - Short Sprint (15/3/10)
  - Deep Work (50/10/20)
  - Quick Study (10/2/5)
- Create unlimited custom presets
- Customize work/break durations
- Set sessions before long break

### Tab 4: 💡 Insights
- **Adaptive Insights & Recommendations**
- Personalized study tips
- Pattern analysis of your sessions
- Best time of day recommendations
- Focus optimization suggestions

### Tab 5: 💾 Data
- **Data Export & Import**
- Export journal entries (JSON)
- Export flashcards (JSON)
- Export settings
- Complete backup functionality
- Import previously saved data
- Merge with existing data

### Tab 6: 🌍 Language
- **Multi-Language Support**
- Choose from 8 languages:
  - English, Spanish, French, German
  - Japanese, Chinese, Portuguese, Russian
- Automatic browser language detection
- Persistent language preferences

### Tab 7: ♿ Accessibility
- **Accessibility Features**
- High contrast mode
- Reduce motion option
- Font size adjustment (4 sizes)
- Sound effects toggle
- Enhanced keyboard navigation
- Screen reader optimization
- Keyboard shortcuts reference

## 🎯 How the Goal System Works

1. **Set Goals** (Settings → Goals tab)
   - Create goal with deadline and target hours
   - Add sub-tasks with due dates

2. **Link Sessions to Goals** (During check-in after each session)
   - Select which goal you worked on
   - Mark tasks as complete

3. **Track Progress** (Settings → Goals tab)
   - See progress bars
   - View "on track" indicators
   - Get daily minute recommendations

4. **Story Integration**
   - Story progression now tied to goal completion
   - Character mood reflects goal progress

## ⌨️ Keyboard Shortcuts

- **Space**: Start/Pause timer
- **R**: Reset timer
- **S**: Skip session
- **J**: Open journal
- **T**: Toggle stats
- **Esc**: Close modals
- **Arrow keys**: Navigate
- **Enter**: Confirm actions

## 💡 Pro Tips

1. **First Time Setup**:
   - Click Settings → Goals → Create your first goal
   - Click Settings → Presets → Choose or create a timer preset
   - Click Settings → Language → Set your preferred language

2. **For Best Results**:
   - Link each session to a goal during check-in
   - Review Adaptive Insights weekly (Settings → Insights)
   - Export your data regularly (Settings → Data)

3. **Demo Mode**:
   - Use quick skip buttons on timer for testing
   - Fast-forward through sessions
   - Perfect for showing clients
